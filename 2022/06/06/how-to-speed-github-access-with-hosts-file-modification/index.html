修改/etc/hosts来加速国内Github访问速度

Posted on 2022-06-06 | In Tool&Tips

对于工程技术人员来说, 经常需要访问Github网站, 但国内有时候访问速度比较慢, 这里面一个很重要的原因是DNS污染导致的访问速度慢. 可以通过修改/etc/hosts改善.

  1. 查找Github 网址对应IP
  2. 修改/etc/hosts

访问站长工具网站-DNS查询页面, 获取github.com和github.global.ssl.fastly.net对应的A record， 选取一个TTL值比较小的IP。

编辑/etc/hosts文件， 将上一步获取的IP地址和对应的域名添加到文件里。例如：

20.205.243.166 github.com
128.242.250.148 github.global.ssl.fastly.net

该操作同时对terminal和浏览器访问加速起作用。
