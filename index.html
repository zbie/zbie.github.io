修改/etc/hosts来加速国内Github访问速度

Posted on 2022-06-06 | In Tool&Tips

对于工程技术人员来说, 经常需要访问Github网站, 但国内有时候访问速度比较慢, 这里面一个很重要的原因是DNS污染导致的访问速度慢. 可以通过修改/etc/hosts改善.

  1. 查找Github 网址对应IP
  2. 修改/etc/hosts

访问站长工具网站-DNS查询页面, 获取github.com和github.global.ssl.fastly.net对应的A record， 选取一个TTL值比较小的IP。

编辑/etc/hosts文件， 将上一步获取的IP地址和对应的域名添加到文件里。例如：

20.205.243.166 github.com
128.242.250.148 github.global.ssl.fastly.net

该操作同时对terminal和浏览器访问加速起作用。

另外，也可以直接访问Github镜像的方式， 例如FastGit, 在git命令中直接替换github.com为hub.fastgit.xyz， 或者git config全局替换：
git config --global url."https://hub.fastgit.xyz/".insteadOf "https://github.com/"
git config protocol.https.allow always

阿里云 maven加速:

修改/usr/share/maven/conf/settings.xml, 将如下内容加入的section下面.

<mirror> <id>aliyunmaven</id> <mirrorOf>*</mirrorOf> <name>阿里云公共仓库</name> <url>https://maven.aliyun.com/repository/public</url> </mirror>
